# Intro de Django con Libro Libre
Esto es una intro a Django 

Crearemos una aplicación web llamada Libro Libre que permitirá hacer:

* Registro de Usuarios (registro, login, logout, cambio de contraseña y visualización y edición de perfil)
* CRUD de libros
* Crud de lugares físicos de intercamvio recomendados
* Catálogo de libros
* Solicitar un libro
* Ver perfil de otros usuarios
